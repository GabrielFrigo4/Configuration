# Scripts

Utilitários diversos organizados por tipo.

| Pasta | Conteúdo |
|:--|:--|
| `PowerShell/` | Scripts `.ps1` (dos2unix, unix2dos, md2pdf) |
| `UnixShell/` | Scripts `.sh` equivalentes |
| `build/` | Scripts de compilação (Aseprite, LibreSprite) |
| `hyper-v/` | Script de configuração Hyper-V |
| `reg/` | Chaves de registro (altgr, emacs, gcc, utf8) |
